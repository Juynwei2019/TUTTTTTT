Python 基礎入門

歡迎來到程式設計的世界！本課程將帶您從零開始學習 Python 程式語言。

第一個 Python 程式


# 這是您的第一個 Python 程式
print("Hello, World!")

在上面的程式碼中，print() 是 Python 的內建函式，用於在螢幕上顯示文字。

變數與資料型態


# 變數宣告
name = "小明"       # 字串
age = 25           # 整數
height = 1.75      # 浮點數
is_student = True  # 布林值

print(f"姓名：{name}，年齡：{age}")
